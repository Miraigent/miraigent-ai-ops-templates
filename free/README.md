# Free Resources

Free resources help teams understand how to prepare for practical AI operations before buying a full template kit.

## Planned Free Resources

- MCP Starter Kit
- AI Readiness Checklist
- Human Review Flow Checklist
- Inquiry Log Before AI Template
- Agent Memories Basic Log Sample

## Purpose

The free layer should:

- build trust
- show the operating model
- help teams avoid unsafe AI workflows
- route serious users to paid implementation kits

Free resources should be useful on their own, but they should not include the full paid product.
