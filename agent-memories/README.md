# Agent Memories

Agent Memories is Miraigent's product direction for turning operational memory into reusable AI workflow assets.

## Concept

Teams do not only need prompts. They need memory:

- what failed
- what worked
- what should not be repeated
- what needs human approval
- what context must be preserved
- what should never be sent to AI

Agent Memories turns these records into practical templates for AI operations.

## Planned Resources

- Basic memory log template
- Decision memory template
- Failure memory template
- Human approval memory template
- Domestic Japanese onboarding materials
- Paid starter kit preview

## 日本語案内

Agent Memoriesは、AI運用で必要になる「判断の記憶」「失敗の記憶」「承認の記憶」「繰り返してはいけないこと」を、再利用できるテンプレートとして整理する考え方です。

国内向けには、まずAI導入前の整理・記録・承認ルールとして伝えます。海外向けには、AI operations resources としてテンプレート化して展開します。
